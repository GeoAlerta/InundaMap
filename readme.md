# InundaMap

InundaMap es un proyecto diseñado para visualizar, gestionar y analizar información relacionada con inundaciones. Este repositorio contiene el código fuente, recursos visuales y configuraciones necesarias para ejecutar el proyecto.

## Tecnologías utilizadas

* **HTML** para la estructura del proyecto
* **CSS** para los estilos y diseño visual
* **JavaScript** para la lógica del frontend
* **YAML** para configuraciones del sistema
* **PNG** e imágenes para elementos gráficos

## Estructura del proyecto

```
InundaMap/
├── index.html
├── css/
├── js/
├── img/
├── config/
└── README.md
```

## Cómo ejecutar el proyecto

1. Clona el repositorio:

   ```bash
   git clone <URL_DEL_REPO>
   ```
2. Accede al directorio del proyecto:

   ```bash
   cd InundaMap
   ```
3. Abre el proyecto en Visual Studio Code:

   ```bash
   code .
   ```
4. Abre el archivo **index.html** en un navegador.

## Flujo de trabajo con GitFlow

Este proyecto utiliza la estrategia **GitFlow**.

* La rama principal es **main**
* La rama de desarrollo es **develop**
* Cada nueva tarea debe hacerse desde una rama **feature**:

  ```bash
  git checkout -b feature/nombre-de-la-tarea develop
  ```
